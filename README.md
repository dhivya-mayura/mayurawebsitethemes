# Style Haven Clothing Website

A modern clothing collection homepage built with HTML, CSS, and JavaScript.

## Files

- `index.html` - Main website page
- `style.css` - Website styling
- `script.js` - Mobile menu, products, and WhatsApp link
- `images/` - Add your own clothing/banner images here

## How to use on GitHub Pages

1. Create a new public GitHub repository, for example `clothing-store`.
2. Upload all files from this folder.
3. Go to `Settings > Pages`.
4. Select `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save.

Your site will be live at:

`https://yourusername.github.io/clothing-store/`

## Important customization

Open `script.js` and replace this phone number:

`919999999999`

with your WhatsApp number including country code.
